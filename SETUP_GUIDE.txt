AI NEWS TOOL — SETUP & USAGE GUIDE
=====================================

WHAT IT DOES
------------
Collects AI news from TechCrunch and TLDR newsletters, summarizes them using Claude,
translates summaries to Chinese, and generates a Word document with two tables:
AI News and AI Fundraising News.

Cost per run: ~$1.00–2.00 | Runtime: ~15–20 minutes


PREREQUISITES
-------------
- Python 3.9+
- A Gmail account subscribed to TLDR AI and TLDR newsletters
- API keys (details in Step 3)


STEP 1 — GET THE PROJECT FILES
-------------------------------
Receive the zipped project folder and unzip it. You should have:

  AI Update/
    tools/
    workflows/
    .env.example
    requirements.txt
    credentials.json    <- you'll need to provide this (see Step 3)


STEP 2 — INSTALL PYTHON DEPENDENCIES
--------------------------------------
Open a terminal, navigate to the project folder, and run:

  python -m venv venv
  source venv/bin/activate       # Mac/Linux
  # OR on Windows:
  # venv\Scripts\activate

  pip install -r requirements.txt
  pip install anthropic           # needed for Claude summarization


STEP 3 — GET API KEYS
----------------------
You need 3 API keys:

  ANTHROPIC_API_KEY   -> platform.anthropic.com          Pay-per-use (~$1–2/run)
  OPENAI_API_KEY      -> platform.openai.com/api-keys    Pay-per-use (~$0.05–0.15/run)
  NEWSAPI_ORG_KEY     -> newsapi.org                     Free (100 req/day)


STEP 4 — SET UP YOUR .env FILE
--------------------------------
Copy .env.example to .env and fill in your keys:

  cp .env.example .env

Open .env and edit:

  ANTHROPIC_API_KEY=your_key_here
  OPENAI_API_KEY=your_key_here
  NEWSAPI_ORG_KEY=your_key_here


STEP 5 — SET UP GMAIL OAUTH (ONE-TIME ONLY)
--------------------------------------------
This lets the tool read TLDR emails from your Gmail inbox.

1. Go to https://console.cloud.google.com/
2. Create a new project
3. Enable the Gmail API
4. Go to APIs & Services -> Credentials -> Create OAuth 2.0 Client ID
   - Application type: Desktop App
5. Download the file and rename it to credentials.json
6. Place credentials.json in the project root folder
7. Run this command once — a browser window will open asking you to log into Gmail:

     python tools/collect_tldr.py --start_date 2026-01-01 --end_date 2026-01-02

8. After logging in, a token.json file is auto-created.
   You won't need to do this again unless the token expires.


STEP 6 — RUN THE TOOL
-----------------------
Set your date range and run all phases:

  START_DATE="2026-03-10"
  END_DATE="2026-03-23"

  # Phase 1: Collect articles (runs in parallel)
  python tools/collect_techcrunch.py --start_date $START_DATE --end_date $END_DATE &
  python tools/collect_tldr.py --start_date $START_DATE --end_date $END_DATE &
  wait

  # Phase 2: Deduplicate
  python3 -c "
  import json
  ai = json.load(open('.tmp/raw_tldr_ai.json'))
  main = json.load(open('.tmp/raw_tldr_main.json'))
  tc = json.load(open('.tmp/raw_techcrunch.json'))
  seen, unique = set(), []
  for a in ai + main + tc:
      url = a.get('url', '')
      if url and url not in seen:
          seen.add(url); unique.append(a)
  json.dump(unique, open('.tmp/classified_articles.json', 'w'), ensure_ascii=False, indent=2)
  print(f'Deduped: {len(unique)} articles')
  "

  # Phase 3: Summarize with Claude
  python tools/summarize_articles.py --provider claude --yes

  # Phase 4: Generate Word doc (with Chinese translation + funding news)
  python tools/generate_word_doc.py --start_date $START_DATE --end_date $END_DATE \
    --articles .tmp/summarized_articles.json --translate


STEP 7 — GET YOUR OUTPUT
--------------------------
The finished Word document is saved to:

  output/AI_News_YYYYMMDD_YYYYMMDD.docx

It contains:
- Table 1 — AI News: Date + Chinese/English summary with article links, sorted oldest to newest
- Table 2 — AI Fundraising News: Company, funding stage, raise amount, investors


COMMON ISSUES
-------------
Problem: insufficient_quota on OpenAI
Fix:     Add billing credits at platform.openai.com/billing

Problem: Gmail token expired
Fix:     Delete token.json and re-run the collect_tldr command — it'll re-authenticate

Problem: Missing API key error
Fix:     Check your .env file has all three keys filled in

Problem: TLDR emails not found
Fix:     Make sure the Gmail account is subscribed to TLDR and emails aren't in spam
